# RCOS (Realtime COuntdown System)

> the greatest modern real-time REST API

A bot (yes, a bot, not an API; don't believe all those Markdown texts starting with `>`) that, every second leading up to a date you specify, sends several thousand requests to a server of your choice with the header: "Yes, Its me!, Yet Another request from RCOS!"

***Strongly not recommended for DDoS attacks! RCOS is humorous software.***
